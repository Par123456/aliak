# سایت مدرسه شهید علی اکبر نجفی کرمانشاه

## راهنمای نصب و استفاده

### فایل‌های سایت

این سایت شامل فایل‌های زیر است که باید همگی در هاست آپلود شوند:

```
school_website/
├── index.html          (صفحه اصلی سایت)
├── admin.html          (پنل مدیریت)
├── styles.css          (استایل صفحه اصلی)
├── admin-styles.css    (استایل پنل ادمین)
├── script.js           (جاوا اسکریپت صفحه اصلی)
├── admin.js            (جاوا اسکریپت پنل ادمین)
├── api.php             (API برای مدیریت داده‌ها)
└── data/               (پوشه داده‌ها - خودکار ساخته می‌شود)
    ├── news.json
    ├── events.json
    ├── announcements.json
    ├── absent.json
    ├── council.json
    └── association.json
```

### مراحل نصب

#### 1. آپلود فایل‌ها

تمام فایل‌ها را در روت هاست خود (معمولاً public_html یا www) آپلود کنید:

- index.html
- admin.html
- styles.css
- admin-styles.css
- script.js
- admin.js
- api.php

#### 2. تنظیمات دسترسی

مطمئن شوید که:
- فایل `api.php` قابل اجرا باشد (chmod 755)
- پوشه `data` با دسترسی نوشتن ساخته شود (chmod 755)

پوشه data به صورت خودکار توسط api.php ساخته می‌شود.

#### 3. دسترسی به سایت

بعد از آپلود، سایت شما در آدرس زیر قابل دسترسی است:
```
https://your-domain.com/index.html
یا
https://your-domain.com/
```

### ورود به پنل مدیریت

#### اطلاعات ورود پیش‌فرض:

```
نام کاربری: admin
رمز عبور: admin123
```

#### مراحل ورود:

1. روی دکمه "ورود مدیر" در بالای سایت کلیک کنید
2. نام کاربری و رمز عبور را وارد کنید
3. بعد از ورود موفق به پنل مدیریت منتقل می‌شوید

### امکانات پنل مدیریت

از پنل مدیریت می‌توانید موارد زیر را مدیریت کنید:

#### 1. اخبار
- افزودن خبر جدید
- ویرایش اخبار موجود
- حذف اخبار

#### 2. رویدادها
- افزودن رویداد جدید
- ویرایش رویدادها
- حذف رویدادها

#### 3. اطلاعیه‌ها
- افزودن اطلاعیه جدید
- ویرایش اطلاعیه‌ها
- حذف اطلاعیه‌ها

#### 4. غایبین
- افزودن دانش‌آموز غایب
- ویرایش اطلاعات غایبین
- حذف از لیست غایبین

#### 5. شورای دانش‌آموزی
- افزودن عضو اصلی
- افزودن عضو علی‌البدل
- ویرایش اطلاعات اعضا
- حذف اعضا

#### 6. انجمن اولیاء و مربیان
- افزودن عضو جدید
- ویرایش اطلاعات اعضا
- حذف اعضا

### ویژگی‌های سایت

#### صفحه اصلی:
- اسلایدر تصاویر
- آمار سریع مدرسه
- نمایش آخرین اخبار
- نمایش رویدادها
- نمایش اطلاعیه‌ها
- امکانات و ویژگی‌های مدرسه

#### صفحه درباره ما:
- معرفی کامل مدرسه
- اهداف و چشم‌انداز
- افتخارات و دستاوردها
- کادر آموزشی
- امکانات آموزشی
- پیام مدیر

#### صفحه غایبین:
- لیست دانش‌آموزان غایب
- فیلتر بر اساس کلاس
- جستجو بر اساس نام

#### صفحه شورای دانش‌آموزی:
- اعضای اصلی شورا
- اعضای علی‌البدل

#### صفحه انجمن اولیاء:
- لیست اعضای انجمن
- اطلاعات تماس

#### صفحه تماس با ما:
- اطلاعات تماس مدرسه
- فرم ارسال پیام
- نقشه موقعیت مکانی

### ذخیره‌سازی داده‌ها

تمام داده‌ها در فایل‌های JSON در پوشه `data` ذخیره می‌شوند:

- **news.json**: اخبار
- **events.json**: رویدادها
- **announcements.json**: اطلاعیه‌ها
- **absent.json**: غایبین
- **council.json**: اعضای شورای دانش‌آموزی
- **association.json**: اعضای انجمن اولیاء

هر تغییری که در پنل ادمین انجام دهید، مستقیماً در این فایل‌ها ذخیره شده و برای همه کاربران قابل مشاهده است.

### پشتیبان‌گیری

برای پشتیبان‌گیری از اطلاعات:

1. وارد پوشه `data` در هاست شوید
2. تمام فایل‌های JSON را دانلود کنید
3. این فایل‌ها را در جای امنی نگه‌دارید

برای بازگردانی:
1. فایل‌های JSON را در پوشه `data` آپلود کنید

### عیب‌یابی

#### مشکل: پنل ادمین باز نمی‌شود
- مطمئن شوید نام کاربری و رمز عبور صحیح است
- کش مرورگر را پاک کنید

#### مشکل: تغییرات ذخیره نمی‌شوند
- بررسی کنید فایل `api.php` قابل اجرا باشد
- مطمئن شوید پوشه `data` دسترسی نوشتن دارد
- لاگ خطاهای PHP را بررسی کنید

#### مشکل: داده‌ها نمایش داده نمی‌شوند
- F5 بزنید یا کش مرورگر را پاک کنید
- بررسی کنید فایل‌های JSON در پوشه data وجود دارند
- Console مرورگر را برای خطاهای JavaScript بررسی کنید

### تغییر رمز عبور ادمین

برای تغییر رمز عبور ادمین، فایل `script.js` را ویرایش کنید:

```javascript
// خطوط 3 و 4
const ADMIN_USERNAME = 'admin';          // نام کاربری جدید
const ADMIN_PASSWORD = 'admin123';       // رمز عبور جدید
```

### سفارشی‌سازی

#### تغییر اطلاعات مدرسه:

فایل `index.html` را ویرایش کنید و اطلاعات زیر را تغییر دهید:
- نام مدرسه
- آدرس
- شماره تلفن
- ایمیل

#### تغییر رنگ‌ها:

فایل `styles.css` را ویرایش کنید و متغیرهای `:root` را تغییر دهید:

```css
:root {
    --primary-color: #2c5aa0;      /* رنگ اصلی */
    --secondary-color: #1e3a70;     /* رنگ ثانویه */
    --accent-color: #ff9800;        /* رنگ تاکیدی */
}
```
