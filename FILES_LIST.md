## بررسی نصب صحیح:

بعد از آپلود، موارد زیر را بررسی کنید:

### 1. آدرس سایت اصلی:
```
https://yourdomain.com/index.html
یا
https://yourdomain.com/
```
باید صفحه اصلی سایت مدرسه نمایش داده شود.

### 2. پنل مدیریت:
```
https://yourdomain.com/admin.html
```
باید صفحه ورود به پنل ادمین نمایش داده شود.

### 3. تست ورود به پنل:
```
نام کاربری: admin
رمز عبور: admin123
```
باید بتوانید وارد پنل مدیریت شوید.

### 4. تست افزودن خبر:
- وارد پنل شوید
- روی "اخبار" کلیک کنید
- یک خبر جدید اضافه کنید
- به سایت اصلی بروید و بررسی کنید خبر نمایش داده می‌شود
